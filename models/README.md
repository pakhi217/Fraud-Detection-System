# Models

This folder stores trained model artifacts.

| File                  | Description                              |
|-----------------------|------------------------------------------|
| fraud_model.pkl       | Best trained model (XGBoost by default)  |
| scaler.pkl            | Fitted StandardScaler for Time & Amount  |
| feature_names.json    | Ordered list of feature names            |
| model_metrics.csv     | Comparison metrics for all trained models|

All files in this folder are generated by:
    python src/train_model.py

The .pkl and .json files are excluded from git (.gitignore).
Commit only model_metrics.csv for documentation purposes.
